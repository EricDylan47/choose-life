# Choose Life - 人生选择模拟器

> "这是一个关于在格拉斯哥贫民区挣扎求生的垃圾故事和垃圾游戏。"

## 游戏简介

你将扮演一个生活在苏格兰格拉斯哥贫民区的吸毒者，在绝望的环境中做出各种选择——是继续沉沦，还是努力挣扎求生？

## 玩法

- **选择行动**: 按数字键 1-6 或点击按钮选择行动
- **继续游戏**: 选择后按回车键或空格键进入下一轮
- **时间限制**: 每轮有25秒的时间限制，超时将随机选择

## 游戏机制

### 属性系统

| 属性 | 说明 |
|------|------|
| 清醒度 | 你的精神状态，归零时游戏结束 |
| 现金 | 你的金钱，没钱时可能债务爆炸 |
| 信誉 | 朋友对你的看法，影响剧情发展 |
| 戒断值 | 毒品依赖程度，影响健康 |
| 焦虑 | 心理压力，影响判断 |
| 绝望 | 心理状态，影响结局 |

### 结局系统

游戏包含多种结局：
- **混沌烂局** - 标准的失败结局
- **被赶出门** - 交不起房租
- **债务爆炸** - 负债过多
- **清醒度归零** - 彻底麻木
- **绝望深渊** - 收到HIV检测结果
- **锒铛入狱** - 犯罪被逮捕
- **过量身亡** - 毒品过量
- **自我救赎** - 成功戒毒
- **人生赢家** - 找到工作，戒掉毒瘾
- **神秘消失** - 神秘的消失

## 安装与运行

### 方式一：直接运行应用

1. 下载或克隆仓库
2. 进入 `dist` 文件夹
3. 双击 `ChooseLife.app` 运行游戏

### 方式二：从源码运行

```bash
# 克隆仓库
git clone https://github.com/EricDylan47/choose-life.git
cd choose-life

# 运行游戏
python3 choose_life_gui.py
```

### 方式三：打包为可执行文件

```bash
# 安装 PyInstaller
pip install pyinstaller

# 打包
pyinstaller --onefile --name ChooseLife --add-data "bgm.wav:." choose_life_gui.py
```

## 操作说明

- **1-6**: 选择行动
- **回车/空格**: 继续下一轮
- **ESC**: 退出游戏
